Minimum spanning tree is used to connect all the nodes/vertices using minimum edges/length 

Example:
In a country we want to lay fibre optic wire so that all the cities are connected, for this we need minimum distance with which all the cities are being connected.


Application:
- Design of netwokrs, power grids, pipeline, transport networks
- Travelling salesman problem
