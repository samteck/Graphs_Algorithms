# SHORTEST PATH GRAPH ALGORITHM

### Shortest Path is used to find shortest distance between 2 nodes/vertices using Dijkstra algorithm

## Example:
We are given distances between 2 discrete cities of a country and we need to find shortest disatnce between any 2 cities (which are not directly connected)

## Application:
- Used in Google maps to find shortest distance between 2 places
- In shppoing mall you need to provide to shortest distance between 2 items to your customer
- How linkedin shows your 1st and 2nd degree connections
